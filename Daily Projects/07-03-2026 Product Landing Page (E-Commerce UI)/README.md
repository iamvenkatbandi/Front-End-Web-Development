# Project Name
**Product Landing Page (E-Commerce UI)**

---

# Project Description

You are building a **modern product landing page for a headphone product** for an **e-commerce company**.

Major platforms like **Amazon, Flipkart, and Apple** use similar landing pages to **highlight products, explain features, and increase conversions**.

Your task is to **design a clean, modern, and responsive product landing page using only HTML and CSS**.

The page should showcase the product clearly while maintaining **good spacing, alignment, and visual hierarchy**.

---

# Learning Goals

This project will help you practice:

- Writing **clean and structured CSS**
- Creating **real-world HTML layouts**
- Designing **modern UI sections**
- Improving **alignment and spacing**
- Using **Flexbox for layout management**
- Applying **consistent typography and color systems**
